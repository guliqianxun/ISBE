# Scope

- 主标的：NVDA（单只股票观察，不做组合）
- Watchlist：TSM / AMD / AVGO / MSFT / GOOG / META / AMZN（产业链 + hyperscaler 客户）
- 大盘 benchmark：NDX（Nasdaq 100）/ SOX（费城半导体指数）

# 隐私边界（红线，spec §4.3）

- ❌ **不**记录任何持仓数据：股数、平均成本、现金额度、盈亏、目标价。
- ❌ digest 不算 P&L、不喊单、不给买卖建议、不给止损位。
- ✅ 仅记录抽象风险偏好（"风险=中"，不带数字）。
- ✅ 信号产出限于：归因、事件 horizon、thesis 状态变化。

# 关注时段

- 美股盘后 22:30 UTC（北京时间次日 06:30 起 30 分钟内）日报。
- 如果有重大事件（财报、Fed、SEC 8-K），日报里突出。
