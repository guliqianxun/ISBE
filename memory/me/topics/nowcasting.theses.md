# 论点 #1
diffusion-based nowcasting 在 lead-time > 90min 仍易 mode-collapse；需更强 conditioning 或物理约束。
- 重要性：mid
- 证据：尚少（待积累）

# 论点 #2
雷达 mosaic 自身的 latency 成为 < 10min lead-time 的瓶颈；模型再快也无用。
- 重要性：low
- 证据：经验观察
